## 📄 MCP 설명 (필수) - 최대 500자

```
"피부과"가 아닌 "아토피 전문 피부과"를 찾아드립니다.

기존 병원 검색의 한계를 해결합니다:
• 피부과 검색 → 미용 피부과도 다 나옴
• 아토피 검색 → 실제 전문 진료 여부 불명확

MediMatch는 AI가 증상을 분석하여 해당 질환을 실제로 전문 진료하는 병원을 매칭합니다.

[주요 기능]
1. 증상 분석: 자연어로 증상 설명 → 적합한 진료과목 추천
2. 전문 병원 검색: 질환명 입력 → 해당 질환 전문 병원 검색
3. 지역별 검색: 17개 시/도, 49개 진료과목 지원

건강보험심사평가원 공식 데이터 기반으로 신뢰할 수 있는 정보를 제공합니다.
```

## 🚀 배포 가이드 (Railway 예시)

### 1. Railway 프로젝트 생성
```bash
# Railway CLI 설치
npm install -g @railway/cli

# 로그인
railway login

# 프로젝트 생성
railway init
```

### 2. 환경변수 설정
Railway 대시보드에서:
- `DATA_GO_KR_API_KEY`: 공공데이터포털 API 키
- `HOST`: 0.0.0.0
- `PORT`: 8000

### 3. 배포
```bash
railway up
```

### 4. 도메인 확인
Railway에서 제공하는 도메인 + `/mcp` 경로를 MCP Endpoint로 등록

예: `https://medimatch-production.up.railway.app/mcp`
